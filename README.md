# VoltStack Framework

## Introduction

VoltStack es un framework PHP modular de alto rendimiento diseñado para ser fácil de usar y extender, a comparacion de otros marcos que requieren una configuracion manual para crear una aplicacion fullstack, VoltStack ofrece una estructura de inicio predefinida que permite a los desarrolladores comenzar a crear su aplicacion rapidamente.

Debido a su estructura modular, VoltStack permite crear desde paginas web, servicios, hasta aplicaciones web de alta complejidad y rendimiento solo utilizando los componentes que necesitas.

## Caracteristicas

- Arquitectura Modular en Componentes y Desarrollo.
- Spa desde el incio.
- Soporte para APIs RESTful.
- ORM avanzado para interaccion con bases de datos.
- Generador de Consultas para SQL.
- Cache y  adaptadores para varios tipos de almacenamiento.
- Sistema de archivos.
- Alto Rendimiento integrando FrankenPHP, OpenSwoole y RoadRunner.
